Definition: speaker.c:12

Initializes the speaker over I2S. The I2S bus is initialized for output with a sample rate of 44100Hz, with a 16-bit depth, and right channel audio. Other configurations requires using the ESP-IDF I2S driver directly.

Note
You must enable the speaker after initializing it with Core2ForAWS_Speaker_Enable(). The speaker cannot be initialized at the same time as the microphone since they both share a common pin (GPIO0). Attempting to enable or use both at the same time will cause the device to hard fault.
Returns
esp_err_t. 0 or ESP_OK if successful.

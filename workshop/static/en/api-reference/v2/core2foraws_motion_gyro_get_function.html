Retrieves the rotational measurements from the inertial measurement unit (IMU) motion sensor's gyroscope.

Example:

This example gets the gyroscope values for the roll, yaw, and pitch rotations, then prints them out to serial output.

#include <esp_log.h>
#include "core2foraws.h"
static const char *TAG = "MAIN_MOTION_EXAMPLE";
void app_main( void )
{
core2foraws_init();
float g_roll, g_pitch, g_yaw;
core2foraws_motion_gyro_get (&g_roll, &g_pitch, &g_yaw );

}

esp_err_t core2foraws_audio_mic_enable(bool state)
Enables or disables the device microphone driver.

Enables or disables the device microphone driver.

Initializes the I2S bus for receiving right-only channel audio, with 16-bit depth, at the sample rate defined by AUDIO_SAMPLING_FREQ.

Note
The microphone cannot be enabled at the same time as the speaker since they both share a common pin (GPIO0). Attempting to enable and use both at the same time will return an error.
Parameters
[in]stateDesired state of the microphone. 1 to enable, 0 to disable.
Returns
esp_err_t.
  • ESP_OK : Success
  • ESP_FAIL : Failed to enable the mic

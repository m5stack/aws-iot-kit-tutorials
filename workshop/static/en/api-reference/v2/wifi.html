Wi-Fi

The Wi-Fi library provides convenience functions to perform opperations to the to 2.4GHz band Wi-Fi radio, including simplified provisioning to networks with WEP, WPA, or WPA-2 Personal encryption protocols. The provisioning over Bluetooth Low Energy (BLE) requires using the accompanying iOS or Android app. The radio is shared with the bluetooth radio and only one can be used at a time. By default all network processing is pinned to Core 0 of the ESP32 and it is recommended to lock affinity to using Core 1 for your application logic if possible.

To enable the BSP hardware features, open the KConfig menu using the command pio run --environment core2foraws --target menuconfig from within the root of the project directory in your PlatformIO terminal window and go to the menu Component Config --> Core2 for AWS hardware features to set the features you want to enable.

Example

The following example initializes the Core2 for AWS, which initializes a number of enabled features, including the Bluetooth/Wi-Fi radio and starts the Wi-Fi provisioning process. After the device is provisioned, it will output to serial the different states of Wi-Fi connectivity. When the device connects to the network, it outputs the SSID of the connected network in the message using the esp_wifi_get_config function.

#include <stdint.h>
#include <freertos/FreeRTOS.h>
#include <freertos/task.h>
#include <esp_log.h>
#include "core2foraws.h"
static const char *TAG = "WIFI_EXAMPLE";
void wifi_event_task( void *pvParams )
{
uxBits = xEventGroupWaitBits(
xEventGroup, /* The event group being tested. */
WIFI_CONNECTED_BIT |
WIFI_DISCONNECTED_BIT |
WIFI_CONNECTING_BIT, /* The bits within the event group to wait for. */
pdTRUE, /* The bits should be cleared before returning. */
pdFALSE, /* Don't Wait for all bits. */
portMAX_DELAY ); /* Wait indefinitely. */
if( ( ( uxBits & WIFI_CONNECTED_BIT ) != 0 )
{
wifi_config_t *wifi_conf;
esp_wifi_get_config(ESP_IF_WIFI_STA, wifi_conf);
ESP_LOGI( TAG, "Connected to Wi-Fi with SSID: %s!", wifi_conf->ssid );
}
else if( ( ( uxBits & WIFI_DISCONNECTED_BIT ) != 0 )
{
ESP_LOGI( TAG, "Disconnected from Wi-Fi network" );
}
else if( ( ( uxBits & WIFI_CONNECTING_BIT ) != 0 )
{
ESP_LOGI( TAG, "Connecting to Wi-Fi network" );
}
}
void app_main( void )
{
core2foraws_init();
xTaskCreatePinnedToCore( &wifi_event_task, // Pointer to the task entry function
"wifiTest", // Friendly name of the task
1024 * 2, // Size of the task stack specified in bytes
NULL, // Pointer to parameters passed to task
0, // Priority the task will run (0 is lowest)
( TaskHandle_t * ) NULL, // Handle that the created task can be referenced
1 // The CPU core to exexute the task
);
}

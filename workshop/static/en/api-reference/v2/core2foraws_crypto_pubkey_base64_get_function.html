Gets the public key that's been paired with the pre-provisioned private key.

Gets the public key that's been paired with the pre-provisioned private key.

The maximum length of the public key is defined in the macro CRYPTO_PUB_KEY_SIZE.

Note
It might take a little time to wake the secure element and retrieve the public key.

Example:

Create a variable named pub_key_str with enough dynamically allocated memory in external RAM to store the public key.

#include <stdint.h>
#include <esp_log.h>
#include "core2foraws.h"
static const char *TAG = "MAIN_ATECC608_EXAMPLE";
void app_main( void )
{
ESP_LOGI( TAG, "\tStarting...");
core2foraws_init();
char *pub_key_str = ( char * )heap_caps_malloc( CRYPTO_PUB_KEY_SIZE, MALLOC_CAP_SPIRAM );
esp_err_t err = core2foraws_crypto_pubkey_base64_get( pub_key_str );
if ( err == ESP_OK )
{
ESP_LOGI( TAG, "\t%s", pub_key_str );
}
free( pub_key_str );
}

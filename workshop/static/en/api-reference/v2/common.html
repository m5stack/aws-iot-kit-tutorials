Common

This library provides shared definitions used by multiple drivers. It does not provide any routines to operate a hardware feature on it's own.

Example

The following example has a function that expects to return an error of type esp_err_t. However, the FreeRTOS API that gets called returns a pdPASS, which has a value of 1 for success and pdFail, which has a value of 0 for failure. Additionally, a function is called to check how much heap is remaining and returns an error if it goes below a threshold of 4096 bytes. The return values are aggregated using a bitwise OR assignment operator. It uses the core2foraws_common_error function to normalize the output to an esp_err_t type. If there is failure on any of the calls, it outputs an error to serial and aborts.

standardize.

#include "esp_log.h"
#include "core2foraws.h"
static const char *TAG = "MAIN_COMMON_EXAMPLE";
static esp_err_t enough_heap_left( void )
{
esp_err_t ret = ESP_OK;
if( esp_get_free_heap_size() < 4096 )
ret = ESP_ERR_NO_MEM;
}
static void common_task( void *pvParams )
{
while( true )
{
ESP_LOGI( TAG, "\tLooping" );
vTaskDelay( pdMS_TO_TICKS( 5000 ) );
}
}
void app_main( void )
{
core2foraws_init();
esp_err_t example_err = ESP_OK;
BaseType_t freertos_task_err;
TaskHandle_t common_task_handle = NULL;
freertos_task_err = xTaskCreatePinnedToCore(
&common_task, // Pointer to the task entry function
"commonTest", // Friendly name of the task
configMINIMAL_STACK_SIZE * 2, // Size of the task stack specified in bytes
NULL, // Pointer to parameters passed to task
0, // Priority the task will run (0 is lowest)
( TaskHandle_t * ) &common_task_handle, // Handle that the created task can be referenced
1 // The CPU core to exexute the task
);
example_err |= !freertos_task_err;
example_err |= enough_heap_left();
if( core2foraws_common_error( example_err ) == ESP_FAIL )
{
ESP_LOGE( TAG, "Application error" );
abort();
}
}

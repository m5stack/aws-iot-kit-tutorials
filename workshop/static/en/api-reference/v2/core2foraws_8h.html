esp_err_t core2foraws_init ( void  )

Initializes enabled hardware features.

If no features are enabled, it does nothing. If the buttons on the touch screen are enabled, it initializes the virtual button driver. If the crypto chip is enabled, it initializes the the secure element. If the display is enabled, it initializes the display and touch screen driver. If the motion sensor is enabled, it initializes the 6-axis IMU w/ temperature driver. If the real-time-clock is enabled, it initializes the RTC driver. If the side RGB LED bars are enabled, it initializes the RGB LED driver. The speaker, microphone (audio) and SD card drivers need to be initialized separately as needed.

Returns
esp_err_t.
  • ESP_OK : Success
  • ESP_FAIL : Failed to initialize one or more features

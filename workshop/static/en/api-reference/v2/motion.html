Motion

The motion library provides convenience functions to interact with the MPU6886 6-axis Inertial Measurement Unit (IMU) with on-chip internal temperature sensor. The core2foraws_motion_init function or core2foraws_init wrapper function configures the IMU with preset parameters using the ESP-IDF I2C driver and the thread-safe ropg i2c_manager. The motion library provides sensor readings from the accelerometer, gyroscope, and internal temperature sensor.

Note
A brief cool-down period (20-100ms) between I2C communications to the MPU6886 is recommended for stability.

To enable the BSP hardware features, open the KConfig menu using the command pio run --environment core2foraws --target menuconfig from within the root of the project directory in your PlatformIO terminal window and go to the menu Component Config --> Core2 for AWS hardware features to set the features you want to enable.

Example

The following example initializes the Core2 for AWS, which initializes a number of enabled features, including the power management unit (PMU) and the motion sensor. From within a FreeRTOS task that loops every ~1s, it retrieves the internal temperature, accelerometer, and gyro data which it stores in the variables temp_c, accel_x, accel_y, accel_z, g_row, g_pitch, and g_yaw respectively. It prints the values using the ESP logger.

#include <stdint.h>
#include <freertos/FreeRTOS.h>
#include <freertos/task.h>
#include <esp_log.h>
#include "core2foraws.h"
static const char *TAG = "MOTION_EXAMPLE";
static void motion_task( void *pvParams )
{
ESP_LOGI( TAG, "\tStarting motion test" );
while ( true )
{
float temp_c;
core2foraws_motion_temperature_get( &temp_c );
ESP_LOGI( TAG, "\tMPU6886 Temperature: %.2fÂ°C", temp_c );
vTaskDelay( pdMS_TO_TICKS( 20 ) );
float accel_x, accel_y, accel_z;
core2foraws_motion_accel_get( &accel_x, &accel_y, &accel_z );
ESP_LOGI( TAG, "\tMPU6886 Acc x: %.2f, y: %.2f, z: %.2f", accel_x, accel_y, accel_z );
vTaskDelay( pdMS_TO_TICKS( 20 ) );
float g_row, g_pitch, g_yaw;
core2foraws_motion_gyro_get( &g_row, &g_pitch, &g_yaw );
ESP_LOGI( TAG, "\tMPU6886 Gyro roll:%.2f, pitch:%.2f, yaw:%.2f", g_row, g_pitch, g_yaw );
vTaskDelay( pdMS_TO_TICKS( 1000 ) );
}
vTaskDelete( NULL );
}
void app_main( void )
{
core2foraws_init();
xTaskCreatePinnedToCore( &motion_task, // Pointer to the task entry function
"motionTest", // Friendly name of the task
configMINIMAL_STACK_SIZE * 3, // Size of the task stack specified in bytes
NULL, // Pointer to parameters passed to task
0, // Priority the task will run (0 is lowest)
( TaskHandle_t * ) NULL, // Handle that the created task can be referenced
1 // The CPU core to exexute the task
);
}

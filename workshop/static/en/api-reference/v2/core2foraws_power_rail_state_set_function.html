Definition: core2foraws_power.h:53

Set the specified power rail state from the power management unit (PMU).

This function is used to directly write the AXP192 PMU register value.

Warning
Manipulating the PMU values is only recommended for advanced users as it can lead to a bricked device.
Parameters
[in]railThe power rail to set the state of.
